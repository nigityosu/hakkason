# このゲームの目標
 - 簡単にクリアできるボスにしてください
# ゲームの内容
    - 道を一直線に進むと、ボスがいます。
    - 横に道を作り、主人公の思い出やボスの思い出を配置してください
    - ボスはそのまま進んでも倒せるようにしてください
    - ボスの思い出は、ボスとの戦いに避けることができるようにしてください
    - 主人公の思い出は、主人公の体力を回復するようにしてください

# クリア条件
 - ボスを倒すこと
 - ボスと和解すること
 - ボスの思い出の数によって、エンディングが変わるようにしてください
 - 主人公の思い出の数によって、エンディングが変るようにしてください
